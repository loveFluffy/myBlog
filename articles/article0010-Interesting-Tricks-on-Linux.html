Recording a 30s movie

By shortcut keys "Ctrl+Alt+Shift+R", we can record a 30s movie. and the same shortcut keys for stoping recording. To change the default length of the movie: gsettings set org.gnome.settings-daemon.plugins.media-keys max-screencast-length 60
